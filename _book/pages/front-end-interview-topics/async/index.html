宏任务和微任务

setTimeout(_ => console.log(4));

new Promise(resolve => {
    resolve();
    console.log(1);
}).then(_ => {
    console.log(3);
    Promise.resolve()
        .then(_ => {
            console.log("before timeout");
        })
        .then(_ => {
            Promise.resolve().then(_ => {
                console.log("also before timeout");
            });
        });
});

console.log(2);
  1. 执行顺序 同步任务=>微任务=>宏任务
  2. new Promise 的 callback 是一个同步任务，只有后面的 then 才是异步(微任务)任务
